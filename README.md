# VKTeams Project

## Overview
Проект для автоматической рассылки сообщений пользователям ВКонтакте.

## File Structure
```
VKTeams/
├── bot/
│   ├── bot.py
│   ├── handler.py
│   └── filter.py
├── config.py
├── commands.py
├── main.py
├── reporting.py
└── README.md
```

## Setup and Usage
1. Убедитесь, что все необходимые зависимости установлены.
2. Запустите проект, выполнив `python main.py`.
